"Pyccoon" is a side-to-side documentation generating system.


It descended from Pyccoon - a Python port of Docco: the original quick-and-dirty, hundred-line-long, literate-programming-style documentation generator. It produces HTML that displays your comments alongside your code. Comments are passed through Markdown, while code is passed through Pygments for syntax highlighting. PyStache is used for page templating.


This website is the result of running Pyccoon against its source.


Pyccoon generates the documentation folder structured correspondingly to the code. To create documentation docs for the project in src folder, run the following:


pyccoon -s src -d docs



Pyccoon is released on GitHub under the MIT license.
